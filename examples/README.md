# Examples

These examples demonstrate how to use the mermaid-ast library. They mirror the code examples in the main README.

## Running Examples

```bash
# Run all examples
bun run examples

# Run a specific example
bun run examples/basic-usage.ts
bun run examples/flowchart.ts
bun run examples/sequence.ts
bun run examples/class-diagram.ts
bun run examples/detect-type.ts
```

## Files

- `basic-usage.ts` - Basic parsing and rendering with auto-detection
- `flowchart.ts` - Flowchart diagram parsing, modification, and rendering
- `sequence.ts` - Sequence diagram parsing and rendering
- `class-diagram.ts` - Class diagram parsing and rendering
- `detect-type.ts` - Diagram type detection
